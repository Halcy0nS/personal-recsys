Primary sources checked:
- GitHub repository README for THUIR/ZhihuRec-Dataset
- arXiv paper 2106.06467

Key findings:
- ZhihuRec-100M contains 99,978,523 impressions, 26,981,583 clicks, 3,899,553 queries, 798,086 users, 554,976 answers, 165,012 questions, 240,956 authors, and 72,318 topics.
- The repository lists file sizes:
  - inter_impression.csv: 2.6GB
  - inter_query.csv: 111MB
  - info_user.csv: 135MB
  - info_answer.csv: 917MB
  - info_question.csv: 14MB
  - info_author.csv: 3.1MB
  - info_topic.csv: 413KB
  - info_token.csv: 409MB
- Summed storage size is about 4.19GB in the listed units.
- Smaller variants also exist:
  - ZhihuRec-20M: 19,999,857 impressions
  - ZhihuRec-1M: 999,970 impressions

Sources:
- https://github.com/THUIR/ZhihuRec-Dataset
- https://arxiv.org/abs/2106.06467
